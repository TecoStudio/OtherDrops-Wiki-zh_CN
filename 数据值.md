您可以在两个地方指定附加数据值：

1. 掉落物中:
```
otherdrops:
    SAND:
        - tool: ALL
          drop: WOOL@RED
```

2. 原始方块说明符号中:
```
otherdrops:
    WOOL@RED:
        - tool: ALL
          drop: DIRT
```

您可以使用的数据值取决于您将数据值绑定到哪个方块。请注意，如果此处未列出，您始终可以使用 Minecraft Wiki 中提供的原始数据值（例如 WOOL@14 和 WOOL@RED 给出相同的结果）。
